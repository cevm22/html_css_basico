Caso del "Titulo de la pagina". En esta situacion se coloca en el CSS un estilo de letra y tamaño en la etiqueta H1 , pero, como este elemento contiene el id="page-title", el navegador le da prioridad a los estilos que contengan este ID el cual también se usa un estilo y tamaño de letra DISTINTO al tag H1.
