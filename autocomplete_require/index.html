Botones

 Existen 2 tipos de botones, el que se utiliza por la etiqueta < input type="submit" >.

El cual se utiliza para formularios en las etiquetas "FORM", ya que limpia todos los campos y envía la información después de activarse. Este boton es menos felxible a la hora de manipularse

Boton genérico

Este tipo de botones son felxibles ya que se pueden utilizar para colocar imagenes y editarlos con màs facilidad, de igual manera puede funcionar como boton tipo submit para formularios, pero debe programarse para agregarle màs funciones
